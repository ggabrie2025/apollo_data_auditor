Executive

For: CEO, CFO — the 30-second view.

  • Global risk grade (A–F) with score trajectory
  • Top 3 risks with financial impact
  • Priority actions summary (P1/P2/P3)
  • Sources scanned + last scan date
Key metric: The global score is a weighted average of the 4 pillar scores — Risk Exposure, Compliance, Data Protection, and Intelligence. Each pillar weighted by regulatory impact.

Risk Exposure

For: CEO, CFO, Consultants — the financial answer to "how much are we exposed?"

  • Financial exposure in € and $ — computed from PII count, source type, GDPR/CCPA penalty formulas
  • Breach Theater — simulates a breach scenario: what fine, what PR cost, what remediation cost
  • PII map — which files/tables have which PII types
  • Toxic combinations — detects high-risk PII co-location (IBAN + SSN in same file)
  • Top risk zones — directories or tables with highest exposure

Compliance

For: DPO, Legal, CISO — answers the auditor's questions before they ask them.

  • GDPR Art.9 — special category data detection (health, biometric, religious)
  • GDPR Art.30 — processing activities completeness
  • GDPR Art.32 — technical security measures (encryption, access control)
  • NIS2 — network and information system security posture
  • SOC2 — availability, confidentiality, security controls
  • CCPA — California consumer rights readiness
  • AI Act — pre-compliance for AI system data requirements
  • Priority actions — P1/P2/P3 with deadlines and financial impact

Data Protection

For: IT leads, MSPs, backup admins — the technical security posture.

  • Backup resilience — frequency, age, coverage, offsite status
  • Encryption coverage — which disks, databases, and cloud storage are encrypted
  • Ransomware readiness — recovery point objective (RPO) simulation
  • Data hygiene — stale files, duplicates, orphaned data
  • Access control — over-permissioned users, admin ratio, dormant accounts

Intelligence

For: CTOs, ESN, AI integrators — is your data AI-ready?

  • AI Readiness score — data quality × classification coverage × utility
  • Data quality metrics — completeness, consistency, freshness per source
  • AI blockers — what prevents AI training: PII contamination, missing labels, schema instability
  • AI Act pre-compliance — Article 10 data governance requirements

Scoreboard

For: Auditors, consultants — full transparency, all metrics, no black box.

  • 71 scores with grade (A–F), value, and formula reference
  • 319 raw metrics (D-IDs) — the data points that feed the scores
  • Full registry with D-ID, source, and computation trace
  • Export to JSON for integration with your own audit tools
All values are computed from your actual scan. Zero hardcoded constants. Zero fake defaults. If a source is not scanned, the corresponding scores are absent — not set to zero.
