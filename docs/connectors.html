Sources

Connectors

11 source types. 44 PII types detected automatically across all of them.

All connectors are read-only. APOLLO never writes, modifies, or deletes data. Only SELECT queries and schema inspection are issued.

Files

Local Files
Windows · Linux · macOS
Scans any local filesystem. Detects PII in CSV, XLSX, PDF, DOCX, TXT, JSON, XML and more. NTFS junction loop detection on Windows.
✓ GA
NFS / SMB Shares
Network shares
Scans mounted NFS and SMB/CIFS network shares. Treats them as local paths — no special protocol required.
✓ GA

Databases

PostgreSQL
Port auto-discovery · asyncpg
Scans all schemas and tables. Samples rows for PII detection. Detects column names + content. Auto-discovers ports 5432, 5433.
✓ GA
MySQL
Port auto-discovery · aiomysql
Scans all databases and tables. Read-only via SELECT. Auto-discovers ports 3306, 3307. MIT license connector (no GPL).
✓ GA
MongoDB
motor async · collections
Scans collections with smart sampling (up to 100K docs). Completeness scoring via aggregation pipeline. Auto-discovers port 27017.
✓ GA
SQL Server
aioodbc · MSSQL
Scans SQL Server databases. Schema inspection via information_schema. Auto-discovers port 1433.
✓ GA

Cloud

OneDrive
Microsoft Graph API · read-only scopes
Scans OneDrive personal and business. Files up to 500MB (SharePoint archives). Detects external shares and PII in documents.
✓ GA
SharePoint
Microsoft Graph API · sites
Scans SharePoint sites and document libraries. Detects external sharing, PII, and document metadata.
✓ GA

Directory

Active Directory
Windows Server AD · LDAP
Scans AD users, groups, and OUs. Detects dormant accounts (90-day threshold), weak passwords, admin ratio, and PII in user attributes.
✓ GA
LDAP (OpenLDAP)
ldap3 · SIMPLE bind
Scans OpenLDAP directories. Users, groups, OUs. Configurable base DN, bind credentials, SSL/TLS support.
✓ GA

ERP / App

Pennylane
ERP · API v2 · 25 req/5s
Scans Pennylane ERP: customers, suppliers, invoices. Detects PII in ERP records (SIREN, SIRET, IBAN, email). Cursor-based pagination.
✓ GA

Infrastructure

Infrastructure Scan
Hardware · OS · Backup · SMART
Collects OS info, RAM, disk usage, SMART status, backup configuration, encryption status. Runs automatically at agent startup. No credentials required.
✓ GA
